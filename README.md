# GameSalesDataAnalysis(1971–2023)

Objetivo:
Este projeto tem como objetivo realizar uma análise exploratória de dados sobre vendas de jogos eletrônicos entre 1971 e 2023, identificando padrões históricos, tendências de mercado e especialização de consoles por gênero.

A análise foi desenvolvida utilizando Excel para tratamento inicial dos dados e Power BI para modelagem e visualização.


Dataset público contendo:

Nome do jogo

Ano de lançamento

Plataforma (console)

Gênero

Vendas por região:

América do Norte (NA)

Japão (JP)

Europa (EU)

Outras regiões

Vendas globais

🎯 Perguntas Respondidas

A análise busca responder às seguintes questões:

📈 Como as vendas evoluíram ao longo dos anos?

🎮 Qual console demonstrou especialização em determinado gênero?

🏆 Quais foram os 10 anos com maior volume de vendas?

👑 Qual foi o jogo mais vendido no período analisado?

📊 Principais Insights

Identificação de ciclos de crescimento e declínio no mercado de jogos.

Consoles com maior concentração em gêneros específicos (ex: esportes, ação, RPG).

Determinação dos anos de maior desempenho comercial da indústria.

Ranking do jogo mais vendido globalmente.

🛠 Tecnologias Utilizadas

Microsoft Excel → Limpeza e organização dos dados

Microsoft Power BI → Dashboard interativo
